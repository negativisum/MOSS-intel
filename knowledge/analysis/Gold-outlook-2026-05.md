# 黄金走势研判 — 2026年5月-2027年

> 分析日期: 2026-05-06 09:30 CST

---

## 一、当前黄金的矛盾信号

黄金正处在一个罕见的"矛盾期"：

| 看涨因素 | 看跌因素 |
|----------|----------|
| 全球地缘战争（中东+欧洲+西半球） | 实际利率急速上升（30年期美债破5%） |
| 去美元化加速（金砖替代体系） | 美元走强（DXY 98.50，突破200日均线） |
| 通胀恐慌（油价$114+） | 巴克莱取消2026年降息预测；卡什卡利"可能加息" |
| 全球央行持续增持黄金储备 | 黄金近期跌2%—技术面短期承压 |

**核心矛盾**：黄金同时在接收"避险买盘"和"利率压制"的信号。哪个更强，决定了方向。

## 二、情景推演

### 情景A：霍尔木兹全面战争 → 黄金飙升至$5,000+（概率 25-30%）

**触发条件**：
- 美伊在霍尔木兹爆发大规模水面交战
- 伊朗再次大规模攻击海湾国家石油设施（沙特被卷入）
- 以色列"抢先动手"攻击伊朗本土核设施
- 油价突破$130-150/桶 → 全球经济滞胀演化为全面衰退

**黄金逻辑**：
- 地缘恐慌压倒一切利率因素 → 黄金作为"终极避险资产"飙升
- 历史上的类似情景：1979年伊朗革命+1980年两伊战争—黄金从$226涨至$850（+276%）
- 1979年美联储基准利率在11-14%，仍然止不住黄金涨势
- 同理：即使美联储不降息，全面战争的恐慌推动黄金突破$5,000

**目标区间**：$5,000-5,500

### 情景B：特习会成功降温 → 黄金回落到$4,000-4,300（概率 30-35%）

**触发条件**：
- 5月14-15日特习会达成某种"中东降级框架"
- 伊朗外长阿拉格齐访华成果转化为实际的停火机制
- 霍尔木兹海峡部分通航恢复
- 油价回落至$95-105/桶

**黄金逻辑**：
- 地缘风险溢价消退 → 黄金从"恐慌定价"回归"利率定价"
- 若同时美联储维持利率或加息预期 → 黄金承压
- 但去美元化+央行购金提供长期底部支撑

**目标区间**：$4,000-4,300

### 情景C：僵持延续 → 黄金维持$4,400-4,800区间震荡（概率 35-40%）

**触发条件**：
- 霍尔木兹"半打半谈"状态持续—Project Freedom继续，伊朗继续骚扰但不全面升级
- 特习会未取得突破也未破裂
- 油价维持在$100-120
- 美联储维持利率不变

**黄金逻辑**：
- 战争风险提供底部 → 利率压力压制顶部
- $4,400-$4,800的宽幅震荡区间
- 央行购金提供结构性买盘支持

**目标区间**：$4,400-4,800

## 三、关键变量和转折点

### 变量1：美联储会不会加息？（最重要）

| 情景 | 概率 | 对黄金的影响 |
|------|------|------------|
| 维持利率不变 | 55% | 中性偏多 |
| 预防性加息（+25bp） | 25% | 短期利空（-$100-200） |
| 因油价通胀被迫连续加息 | 15% | 大幅利空（-$300-500） |
| 战争迫使紧急降息 | 5% | 极度利多（+$500+） |

**判断**：卡什卡利的"不能排除加息"目前仍是口头干预。但若油价持续$120+并推高核心PCE超过4%，美联储可能被迫行动。**这将是黄金最危险的时刻**——加息预期会剧烈压制金价，即使地缘风险仍在。

### 变量2：去美元化进程

- 中国减持5000亿美元债 → 人民币跨境结算占比升至30%
- 阿联酋退出OPEC后暗示可能用人民币结算石油
- 金砖支付系统推进中
- **每一条去美元化的新闻都是黄金的结构性利多**，但这个因素在数月时间尺度上缓慢积累

### 变量3：央行购金

- 中国央行连续18个月增持黄金（预计）—这是黄金的"政策底"
- 全球央行2024-2025年购金量创纪录
- 只要央行持续购金，黄金很难大幅下跌

## 五、综合判断

### 中期走势（3-6个月）：偏多，但路径波动

```
基准目标区间：$4,500-5,000
上行风险：$5,000-5,500（全面战争情景）
下行风险：$4,000-4,300（外交突破情景）
```

### 关键判断

1. **地缘风险溢价仍是黄金的主要定价因素**，而非利率。$4,560的金价里，大约$800-1,000是战争溢价。战争不结束，黄金难大跌。

2. **特习会是近期最大的方向性催化剂**。黄金的短期走势很大程度上取决于这次会议的结果。

3. **美元走强+加息预期是"假利空"**——在全面战争的恐慌面前，利率因素会被淹没。但如果战争风险消退，利率因素就会重新主导。

4. **央行购金是黄金的"安全网"**——即使地缘风险消退，全球央行（特别是中国、波兰、印度）的持续购金也会在$4,000附近提供强支撑。

5. **最大的黑天鹅**：如果美联储真的被迫加息（卡什卡利情景），而地缘风险又因为外交突破消退——双重利空叠加，黄金可能测试$3,800-4,000。但这是小概率（~15%）。

### 一句话结论

**只要霍尔木兹海峡仍在打仗、油价仍在$100上方，黄金就难以下破$4,000。特习会如果成功降温中东，黄金可能回落10-15%；如果失败，黄金可能突破$5,000。当前宜持金不动，等5月14-15日特习会结果再做方向性判断。**
