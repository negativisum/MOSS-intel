## 〇、被低估的变量：民族情绪 —— 当日本修宪废除第九条

之前框架的核心缺陷：**过度依赖理性计算，低估了中国国内政治压力对决策的约束力。**

### 这不是"想不想打"的问题——是"敢不敢不打"的问题

让我们推演日本修宪成功后的中国国内反应：

**第一波（修宪公投通过当天）**：
- 社交媒体："日本撕毁和平宪法 = 撕毁投降书 = 军国主义复活"
- 民族主义舆论统一口径："1945年的胜利被否定了"
- 历史记忆被激活：南京、731、慰安妇 → 反日情绪全面爆发
- 抵制日货运动自发重启，规模将远超2012年

**第二波（政府反应的窗口极其狭窄）**：
- 任何一个中国领导人面对日本废除第九条，**不作为的代价可能高于行动的代价**
- 这不是传统地缘政治计算——这是执政合法性问题
- 如果日本公然否定战后秩序而中国不做出"历史级别的回应"→ 国内的质疑将从"为什么不动武"升级为"谁在出卖二战胜利成果"
- 在中共的政治逻辑里，维护"抗日战争胜利成果"的政治正确性是不可挑战的

**第三波（历史类比的心理重量）**：
- 1931年九一八 → 不抵抗 → 民族创伤至今未愈
- 2027年日本修宪废除第九条并正式将"台湾有事"写入国家安全战略
- 如果中国"不动武"→ 历史叙事将被改写为"又一次放任军国主义复活"
- **没有领导人能承担这个历史评价**

### 所以问题不是"中国会不会对日动武"

我之前的分析把问题框定为"中国会不会对日动武"——这是错误的框定。正确的框定是：

**日本修宪废除第九条 → 中国必须作出"历史级别的回应"**

这个回应大概率不是直接对日开战（中日全面战争不是任何一方的首选）。更可能的回应层级：

| 回应层级 | 具体措施 | 必然性 |
|---------|---------|--------|
| 法律层面 | 联合国推动重申波茨坦公告第八条 + 宣布《中日和平友好条约》失效 | 几乎是自动的 |
| 经济层面 | 全面经济脱钩，稀土+所有战略物资断供 | 极高 |
| 军事层面 | **对台实施"准封锁"或直接军事行动** | 高 |
| 外交层面 | 驱逐大使、外交降级 | 极高 |

### 修正后的概率

| 情景 | 修正前 | 修正后 | 理由 |
|------|--------|--------|------|
| 日本修宪后一年内两岸发生军事冲突 | 25-35% | **50-65%** | 民族情绪+政治必要性双重约束 |
| 其中：对台准封锁（不开枪，禁航/禁飞/检查） | - | 40-50% | 最小满足"做了历史性回应"的要求 |
| 其中：对台全面军事行动（登陆/打击） | - | 10-15% | 视美日反应决定是否升级 |

### 一条需要补充的关键逻辑

中国到目前为止的策略——法律战（开罗宣言碑）+ 经济战（稀土断供）+ 外交战（分化日美）+ 认知战（放大日本国内反战声音）——本质上是在 **"在修宪成功之前破坏修宪"** ，而不是在"为修宪后的动武做准备"。

但如果这些手段都没能阻止修宪 → 那中国确实会被逼到墙角。而在墙角里，民族情绪会把"不打"从审慎的选择变成政治上的不可能。

## 一、回到你的原始问题：为什么现在不主动出击

基于以上框架，这个问题的答案更清晰了：

### 中国不是在"等"，而是在"阻止"

每一层当前策略都是在**让修宪不成功**：

| 策略 | 目标 | 进度 |
|------|------|------|
| 开罗宣言碑 | 在国际法层面锁定"日本修宪=撕毁投降书" | ✅ 已落地（5月3日） |
| 稀土断供 | 让日本经济承受修宪的代价 | 🔄 进行中，1年期效果约-1.3~3.2% GDP |
| 外交分化 | 让东南亚/欧洲国家不愿支持日本修宪叙事 | 🔄 高市小泉接力出访 vs 中国影响力 |
| 认知作战 | 放大日本国内反战/反修宪民意 | 🔄 修宪公投需过50%，仍有运作空间 |
| APEC设条件 | 留一个外交台阶让日本后退 | 待验证 |

### "主动出击" = 帮日本修宪

如果中国现在就对台动武：
- 日本修宪公投通过率将从50-60%飙升至80%+
- 美国在亚太的盟友将立即团结
- 中国从"秩序维护者"变为"秩序破坏者"

**现在的不打，恰恰是为了让2027年不必打。**

## 三、结论

**你问"如果和平宪法第九条废除，不会对日动武吗？"**

我的判断是：

1. **中国不会"对日动武"**——中日全面战争是所有人的噩梦，且会立即触发美日安保条约第五条
2. **但中国必然做出"历史级别的回应"**——而这个回应最可能的军事形式是**对台湾而不是对日本**采取行动
3. **逻辑链**：日本修宪 = 日本否定二战后秩序 = 作为二战战胜国的中国必须捍卫战后秩序 → 而战后秩序的核心成果之一就是台湾归还中国 → 对台采取军事行动 = 以实际行动宣告战后秩序不可动摇
4. **为什么打台湾而不是打日本**：打台湾在法律上是"维护战后秩序"，打日本在法律上是"发动战争"。前者有《开罗宣言》和《波茨坦公告》背书，后者没有。

### 一句话总结

**日本修宪废除第九条不会导致中国对日宣战。但极可能导致中国对台动手。不是因为中国"想打"，而是因为届时中国"不打"的政治代价将大于"打"的军事代价。**
