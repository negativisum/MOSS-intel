## 二、核心困境：通胀 vs 衰退，美股两头挨打

当前美股处在最尴尬的位置——**"stagflation lite"（轻量版滞胀）**：

```
油价$114+  ──→ 输入性通胀上升 ──→ 美联储不敢降息
                                    │
                                    ▼
                           企业成本上升 + 借贷成本高企
                                    │
                                    ▼
                           盈利预期下调 ──→ 股市承压
```

但与此同时——油价上升对能源板块是利多（能源板块+0.85%）、AI/科技板块有自己的叙事逻辑——所以**不是全面崩盘，而是结构分化**。

## 三、情景推演

### 情景A：霍尔木兹全面战争 → 美股剧烈下跌（15-25%）（概率 25-30%）

**触发链**：
1. 美伊在霍尔木兹爆发大规模水面交战 + 以色列攻击伊朗本土
2. 油价飙升至$130-150/桶
3. 全球供应链第二次断裂 → 通胀全面爆发 → 经济衰退不可避免
4. 美联储被迫在"加息抗通胀"和"降息救经济"之间二选一（两难）

**美股路径**：
- 第一阶段：能源板块暴拉（+10-15%）+ 其他所有板块暴跌
- 第二阶段：能源也撑不住（需求崩塌预期）
- 标普500目标：5,500-6,000（从7,208下跌15-20%）
- 纳指科技股承压更重（高估值+高利率双重打击）

**历史类比**：1973年石油危机 — 标普500跌48%。不过当前不像1973年那么极端，因为美国现在是净石油出口国（页岩油），能源板块反而受益。

### 情景B：特习会成功降温 → 美股强劲反弹（+8-12%）（概率 30-35%）

**触发链**：
1. 特习会达成中东降级框架 + 贸易战缓和
2. 油价回落至$95-105
3. 30年期美债收益率从5%回落至4.5-4.7%
4. 美联储重新获得降息空间（至少不会加息）
5. 衰退恐慌消退

**美股路径**：
- 全面反弹，金融/科技/消费全线领涨
- 标普500目标：7,500-7,800
- 纳指弹性最大（降息预期重燃）

### 情景C：僵持延续 → 震荡偏弱（-5%到+5%）（概率 35-40%）

**触发链**：
1. 霍尔木兹"半打半谈"持续
2. 油价在$105-120之间波动
3. 30年期美债在4.8-5.2%区间震荡
4. 美联储按兵不动
5. 美股在每个"希望"和"失望"之间来回摇摆

**美股路径**：
- 标普500在6,800-7,500之间宽幅震荡
- 板块轮动加剧：能源/军工 vs 科技/消费的跷跷板
- 个股分化严重：盈利不达预期的公司被重罚

## 四、关键变量

### 变量1：油价 — 美股的第一驱动力

| 油价区间 | 对美股的影响 |
|---------|------------|
| $90-100 | 利多 — 复苏交易回归 |
| $100-115 | 中性偏空 — 当前区间 |
| $115-130 | 利空 — 成本压力传导 |
| **>$130** | **灾难级利空 — 滞胀恐慌全面爆发** |

### 变量2：10年期美债收益率 — 估值的锚

当前10年期约4.40%，30年期破5.0%。若30年期持续在5.0%上方：
- 科技股估值（远期现金流折现）受压制
- 高杠杆企业压力增大
- 股息股吸引力相对上升

### 变量3：美联储政策路径 — 最大悬念

| 路径 | 概率 | 对美股影响 |
|------|------|-----------|
| 全年不降息 | 55% | 偏空，但已被部分定价 |
| 降息1-2次 | 20% | 强利多，未被定价 |
| 被迫加息 | 15% | 灾难级利空 |
| 紧急降息（战争/危机） | 10% | 短期利多但长期空（为何而降？） |

## 七、综合判断

### 基准情景：震荡偏弱，关键在特习会

```
标普500 3-6个月目标：6,600-7,500（当前7,208）
方向：偏下行，但无系统性崩盘
特习会是近期最大的方向性变量
```

### 关键判断

1. **美股现在不是由"AI叙事"驱动，而是由"油价叙事"驱动。** 当油价$114+时，AI再惊艳也无法抵消整个经济体的成本冲击。

2. **30年期美债突破5%是一个严重的预警信号。** 这不是普通的市场波动——这意味着债券市场在定价"通胀可能持续失控"。如果这个趋势延续，美股将面临"股债双杀"。

3. **特习会对美股的短期影响可能比美联储决议更大。** 如果特习会释放中东降级信号，美股单周可能反弹5-8%。如果失败，道指可能测试47,000。

4. **能源板块是目前唯一确定性较高的方向。** 无论战争升级还是僵持，$100+的油价都是能源公司的利润保证。但如果外交突破且油价暴跌，能源板块将面临剧烈回撤。

5. **最大的尾部风险（但概率较低）**：美联储被迫加息。这将是美股的"雷曼时刻"——市场完全未定价这个情景。

### 一句话

**美股已经从"AI驱动的牛市"转入"战争驱动的震荡市"。5月14日特习会之前，任何抄底或追涨都面临外部事件风险。特习会之后，如果中东降温 + 美联储维持利率不变，美股有望迎来Q3反弹；如果中东恶化 + 油价破$130，美股将进入真正的熊市。**
