## 二、会谈主要议题推演

### 议题1：伊朗石油——核心中的核心

**背景**：
- 中国进口伊朗石油日均138万桶（占伊朗出口80%+）
- 美国刚制裁5家中国企业→中国立即发布阻断禁令
- 美军今天(5.4)出动1.5万人疏导霍尔木兹海峡
- 伊朗14点方案已提交，美方已回复，但特朗普"无法想象可接受"

**特朗普可能的要价**：
> "停止进口伊朗石油，作为回报取消25%关税+解除涉伊制裁"

**中方底线**：
- 完全停止进口→不可能（能源安全+战略自主）
- 可考虑：公开承诺"配合谈判"，私下维持进口量
- 可考虑：以"第三方贸易"形式继续（中资香港/新加坡公司名义）

**最可能结果**：模糊妥协——中方承诺"支持外交解决伊朗问题"但不明确承诺停止进口，美方暂缓解除部分制裁但保留施压空间

### 议题2：贸易与关税

**背景**：
- 特朗普本周威胁将欧盟汽车关税提至25%
- 美国核心通胀3年最高→继续加关税可能反噬
- 中国阻断禁令=中美法律战已升级

**特朗普的可能要价**：
> "中国大幅扩大自美进口，换取关税暂停甚至降低"

**中方策略**：
- 扩大采购美国能源（LNG/石油，弥补伊朗油潜在减少）
- 采购波音飞机（波音因中美摩擦积压大量订单）
- 采购农产品（传统项目，政治象征意义大）
- 但不会签"不平等条约"

### 议题3：台海/南海

**背景**：
- 美菲NMESIS部署巴丹群岛（距台140公里）
- 日本解禁杀伤性武器出口→本周向菲无偿转让护卫舰
- 日本首相+防长接力出访东南亚

**特朗普的可能要价**：
> "中国约束在台海/南海的军事活动，美国约束对台军售节奏"

**中方策略**：
- 台海=红线，不接受任何交换
- 南海=可讨论行为准则
- 要求美方：停止在日本部署中导、约束菲律宾挑衅

### 议题4：俄乌战争

**背景**：
- 俄罗斯表态倾向和平协议
- 乌克兰纵深打击能力提升
- 特朗普此前要求俄50天内结束战争

**中方筹码**：
- 对俄有独特影响力
- 可扮演"和平调解人"
- 换取美国在其他议题上的让步

### 议题5：科技与供应链

- 芯片出口管制→中方要求放松/美方大概率维持
- 硫酸禁令→中方可能作为谈判筹码
- AI/量子计算→各自为政，不太可能达成协议

## 四、🇨🇳 中方最优策略——"四项原则+三步走"

### 四项原则

**1. 不对抗，但保留反制能力**
- 接受对话但不接受威胁
- 阻断禁令已经展示→"美国制裁不好使"
- 保持军事准备但不能被拖入军备竞赛

**2. 不选边，保持战略自主**
- 伊朗石油：不完全停止，不公开挑衅
- 俄乌：劝和促谈，但不出卖俄罗斯
- 不加入任何一方阵营

**3. 不急于求成，等待窗口期**
- 美国内部困境：通胀+清洗+战争+欧洲裂痕→时间在中国这边
- 伊朗战争消耗美国国力→越久对美国越不利
- 特朗普访华本身说明→美国需要中国

**4. 不孤军，巩固多边阵线**
- 东盟峰会(5.8-9)是重要平台
- 金砖/上合/中俄伊→不做美国的"唯一对话者"
- 多极化=中国最大战略资产

### 三步走操作路径

**第一步：访华前（5月4-13日）——造势与设槛**

| 行动 | 目的 |
|------|------|
| 外交部释放"欢迎对话但不接受施压"信号 | 设定谈判基调 |
| 在东盟峰会(5.8-9)展示区域领导力 | 多边平台增加筹码 |
| 商务部继续执行阻断禁令→检验美国反应 | 展示决心 |
| 扩大与沙特/阿联酋/卡塔尔能源合作 | 降低伊朗石油依赖度，增加谈判灵活性 |
| 不主动升级台海/南海局势 | 不给美国"中国威胁论"口实 |

**第二步：访华期间（5月14-15日）——交换与包装**

| 议题 | 中方策略 |
|------|----------|
| 伊朗石油 | 承诺"支持和平解决"→模糊处理，不承诺停止进口 |
| 贸易 | 宣布扩大自美进口（能源/飞机/农产品），量化目标 |
| 台海 | 重申一个中国→要求美方"以实际行动恪守承诺" |
| 俄乌 | 提出中国版"和平方案"→包装为全球公共产品 |
| 气候/卫生 | 寻找非争议性合作领域→为峰会制造正面成果 |

**关键话术设计**：
> "中美关系不是谁输谁赢的零和博弈。我们愿意扩大合作面，但核心利益不容交易。"

**第三步：访问后（5月16日后）——评估与调整**

| 场景 | 行动 |
|------|------|
| 如果美方实质让步（减关税/解制裁） | 逐步扩大合作领域，同时在南海保持克制 |
| 如果美方口惠实不至 | 加速去美元化→扩大人民币石油结算→深化中俄伊合作 |
| 如果美方继续施压 | 升级阻断机制→扩大到更多领域→联合金砖国家反制 |

## 五、特朗普此行的内部政治逻辑

**为什么特朗普现在需要访华？**

1. **伊朗战争陷入僵局**：65天战争无决定性胜利→需要中国配合切断伊朗经济
2. **国内压力山大**：通胀3年最高+航空业"阵亡"+五角大楼内乱→需要"外交胜利"
3. **欧洲关系恶化**：对欧盟加关税+对教皇开骂→需要另一个"大国对话"转移焦点
4. **中期选举(11月)**：需要"可量化"的外交成果
5. **个人风格**：喜欢"大交易"→访华=可以宣布"历史性协议"

**这是中方的战略机遇**：
- 美国有求于中国→要价空间大
- 但不等于可以"搞定"特朗普——他可能签完协议就撕毁

## 七、核心判断

1. **特朗普访华的本质**：不是"访问"，是"求助"——美国在伊朗战争中需要中国
2. **中方的优势**：多线博弈中的"关键第三方"——对伊朗、俄罗斯、朝鲜都有影响力
3. **最优策略**：以时间换空间，以多边对单边，以合作换缓和，但不做战略性让步
4. **底线不可交易**：台湾问题、发展权、伊朗石油的"正常贸易"权利
5. **最大风险不是特朗普**：而是以色列可能在访华前/后单独行动，打乱所有谈判节奏
