## 🔗 金融传导链：关键关联

### 情景1：美伊MOU达成
- 霍尔木兹重开 → 油价急跌至$80-85 → 通胀预期降温 → 美联储降息空间打开 → 美债收益率下行 → 黄金承压($4,200-4,400) → 美股反弹 → 新兴市场资本回流

- 海峡封锁持续 → 油价破$120 → 全球通胀飙升 → Fed被迫加息 → 美债收益率飙升至4.5-5% → 美股暴跌 → 新兴市场货币贬值 → 粮食危机加剧 → 全球衰退风险

- 欧洲能源压力缓解 → 全球风险偏好改善 → 资金从避险资产流出 → 黄金短期承压 → 欧洲资产受追捧
